A command line Implementation of FreeCell in [Hot Cocoa Lisp](https://github.com/olleicua/hcl).

Usage:

    npm -g install hot-cocoa-lisp
    git clone git@github.com:olleicua/FreeCell.git
    cd FreeCell
    hcl -nu game.hcl
